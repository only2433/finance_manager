# finance_manager

- Firebase Auth 를 이용한 계정 생성 및 로그인
- Firebase Database 및 Storage를 이용 하여, 각각의 계정에 대한 정보 및 가계부 아이템 정보를 Save & Delete & Load
- GetX를 이용한 Binding 및 Stream
